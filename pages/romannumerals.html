Roman Numerals

Roman numerals are used to analyze chords - 'Roman Numeral Analysis' (RNA).
There are three main conventions: classical, jazz (Mark Levine), and the Nashville Number System (though technically not roman numerals).

I generally work with more modern music than classical, so I will primarily be using the jazz conventions.
It is important to follow the conventions of whatever circle you happen to be in.
"Bad conventions are better than no conventions at all."
I will not be going to deep into each of these conventions because that is a chapter all in itself.
The main difference between jazz and classical is that jazz uses all upper case romans.

Roman numerals are used to abstract chord progressions, the same way scale degrees are used to abstract melodies.
